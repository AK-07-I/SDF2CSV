ABOUT - SDF2CSV


What is SDF file format?

SDF (structure data file) is one of a family of chemical-data file formats developed by MDL. It consists of a series of molfiles containing structural information joined together, together with some additional information about the compounds. They are frequently used for sharing libraries of compound structure data.

What is a csv file?

A CSV (Comma Separated Values) file is a plain text file that stores data by delimiting data entries with commas. CSV files are often used when data needs to be compatible with many different programs. CSVs can be opened in text editors, spreadsheet programs like Excel or libreoffice, or other specialized applications.

SDF2CSV

SDF2CSV is a web application developed using Flask and RDkit libraries, to extract the contents from SDF MOL file to a CSV file.
